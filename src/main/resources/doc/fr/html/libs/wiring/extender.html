Comportement

L'extenseur de bits modifie la largeur de bit d'une connexion. Si sa largeur est réduite, elle est simplement tronquée pour conserver les bits de poids le plus faible. Si sa largeur de bit est augmentée, les bits de poids faible ne sont pas modifiés, et vous avez le choix pour les bits de poids fort supplémentaires: ils peuvent êtres soit tous à 0 soit tous à 1 ou tous égale au bit de signe (le bit le plus élevé de l'entrée) ou encore êtres déterminés par une entrée de commande supplémentaire.

Pins

Bord ouest :Entrée, la largeur de bit est définie par l'attribut Largeur de bit en entrée

L'entrée multibits dont la valeur doit être transformée.

Bord Est :Sortie, la largeur de bit est définie par l'attribut Largeur de bit en sortie

La sortie transformée.

Bord Nord :Entrée, largeur de bit 1

Spécifie quels devraient être les bits supplémentaires dans la sortie. Cette broche est disponible uniquement lorsque l'attribut Type d'extension est Entrée.

Attributs

Lorsque le composant est sélectionné ou ajouté, 0 à 9 modifie l'attribut Largeur de bit en entrée,Alt-0 à Alt-9 modifie l'attribut Largeur de bit en sortie

Largeur de bit en entrée
Décrit plus haut.
Largeur de bit en sortie
Décrit plus haut.
Type d'extension
Dans le cas d'une extension de la largeur, cette option permet de déterminer l'état des bits supplémentaire. Si zéro ou un, les bits supplémentaires seront égale à 0 ou à 1 en conséquence. Si Signe , les bits supplémentaires sont ajustés pour correspondre au bit de poids fort de l’entrée. Et si Entrée, La valeur d'une seconde entrée de un bit sur son côté est utilisée pour définir les bits supplémentaires.
