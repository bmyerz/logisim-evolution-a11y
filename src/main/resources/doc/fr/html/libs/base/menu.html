Outil Menu

Librairie: Base
Introduction: 2.0 Beta 1

Comportement

L'outil de menu permet à l'utilisateur de créer un menu contextuel pour les composants existants. Par défaut, un clic droit ou un contrôle-clic sur un composant fait apparaître ce menu contextuel; cependant, l’onglet Souris du options de projet permet à un utilisateur de configurer les boutons de la souris pour travailler différemment.

Le menu local de la plupart des composants comporte deux éléments.

  • Supprimer: Supprime le composant du circuit.
  • Afficher les attributs: place les attributs du composant dans la table des attributs de la fenêtre, afin que les valeurs d'attribut puissent être affichées et modifiées.

Pour certains composants, cependant, le menu contient des éléments supplémentaires. Les sous-circuits (c'est-à-dire les cas d'utilisation d'un circuit en tant que "boîte noire" dans un autre) en sont un exemple: en plus des deux éléments ci-dessus, le menu contextuel comprend un autre élément.

  • Vue xxx: Change l'affichage pour présenter le circuit spécfié dans le menu. Les circuits seront éditables. Ces circuits sont un élément d'une structure hierachique plus grande. Voire Déboguer un sous-circuit de la section Guide de l'utilisateur. Changes the circuit layout being viewed and edited to be the subcircuit's layout instead. The values seen in the layout will be part of the same hierarchy as those of the supercircuit. (See the `Debugging subcircuits' section of the User's Guide).

D'autres composants peuvent également étendre le menu contextuel. Dans les bibliothèques intégrées de la version actuelle de Logisim, les seuls composants de ce type sont RAM et ROM .
