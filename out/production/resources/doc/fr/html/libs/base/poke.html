Comportement

L'outil Pousser permet de manipuler les valeurs actuelles associées aux composants. Le comportement précis de l'outil Pousser varie en fonction du composant cliqué. Ce comportement est documenté dans la section Comportement de l'outil Pousser de chaque composant individuel. Les composants suivants prennent tous en charge l'outil Pousser.

De même, cliquer sur un segment de fil à l’aide de l’outil Pousser affiche la valeur transportée par le fil, comme décrit sur la page de l'outil câblage.

Dans le mode dessins en cliquant sur un composant prenant en charge l'outil Pousser, vous afficherez les attributs de ce composant.
