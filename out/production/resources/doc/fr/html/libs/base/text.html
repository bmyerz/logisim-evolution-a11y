Outil Text

Librairie: Base
Introduction: 2.0 Beta 1

Comportement

L'outil texte vous permet de créer et de modifier des étiquettes associées à des composants. Les composants qui prennent en charge les étiquettes sont indiqués dans la section «Comportement de l'outil de texte» de leur documentation. À partir de la version actuelle, les composants suivants des bibliothèques intégrées prennent en charge les étiquettes.

Base library Pin
Clock
Label
Probe
Memory library D/T/JK/SR Flip-Flop
Register
Counter
Shift Register
Random
Input/Output library Button
LED

Pour les composants qui peuvent prendre une étiquette sans en avoir aucun, vous pouvez cliquer n'importe où dans le composant pour ajouter une étiquette. S'il y a déjà une étiquette, vous devez cliquer dans l'étiquette. Si vous cliquez sur un point pour lequel aucune étiquette ne doit actuellement être modifiée, Logisim lance l'ajout d'un nouveau composant Label.

Dans la version actuelle de Logisim, les fonctionnalités d'édition de texte sont encore assez primitives. La sélection d'une région de texte dans une étiquette est impossible. Il n'y a aucun moyen d'insérer un saut de ligne dans une étiquette.

Attributs

Les attributs de l'outil sont les mêmes que pour le composant d'étiquette . Ces attributs n'ont aucun effet lors de l'édition de l'étiquette sur un composant existant, mais ils sont transmis aux étiquettes créées à l'aide de l'outil de texte.

En cliquant sur un composant prenant en charge l'outil Texte, les attributs de ce composant s'affichent.
