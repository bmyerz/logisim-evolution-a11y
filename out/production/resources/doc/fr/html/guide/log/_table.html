The Table tab

L'onglet Table affiche le journal actuel sous forme graphique. The Table tab displays the current log graphically.

Le tableau contient une colonne pour chaque composant de la sélection. Chaque ligne du tableau affiche un instantané de la simulation une fois la propagation des valeurs terminée. Toutes les lignes en double ne sont pas ajoutées au journal. Notez que seules les 400 dernières lignes sont affichées. Certaines lignes peuvent avoir des entrées vides si le composant correspondant n'était pas dans la sélection au moment où la ligne a été calculée. The table contains a column for each component in the selection. Each row in the table displays a snapshot of the simulation after a propagation of values has completed. Any duplicate rows are not added into the log. Note that only the most recent 400 rows are displayed. Some rows may have empty entries if the corresponding component was not in the selection at the time that the row was computed.

La table affichée est à titre indicatif seulement; ce n'est pas interactif. The displayed table is for review only; it is not interactive.
